HYPERLINK, HTML LINK SYNTAX, & TARGET ATTRIBUTE


HYPERLINK

Links are found in nearly all web pages. Links allow users to click their way from page to page.

A link does not have to be text. A link can be an image or any other HTML element.

Source Code:
< a href ="https://www.philsca.edu.ph"> Philippine State College of Aeronautics < / a >

Example Output:
Hello world! I am Daniela Dean, a first-year college student, and I am currently studying at Philippine State College of Aeronautics. This is an examle of an website link.


HTML LINK SYNTAX

When inserting a link for the user to interact with, the encoder must use the "< a >" element with the href attribute which indicates the link's destination.

Source Code:
< a href ="https://www.instagram.com/mochieru_/"> My Instagram < / a >

Example Output:
Try visiting my Instagram page. @mochieru_


HTML LINK TARGET ATTRIBUTE

The target attribute specifies where to open the linked document.

target attribute can have one of the following values:

_self - Default. Opens the document in the same window/tab as it was clicked.

_blank - Opens the document in a new window or tab.

_parent - Opens the document in the parent frame.

_top - Opens the document in the full body of the window.

Example:

Visit the Artist Website (G)I-DLE

If target tag ="_blank" is used, the user will be redirected to a new browser window or tab.

Visit the Artist Website (G)I-DLE

The target tag ="_self" is the default action, if the link is clicked it will redirect the user using the current tab.

Parent HTML document

If target tag ="_parent" is used, when clicked, the user will be redirected to another HTML file. As long as you have the same file in your computer.

CREATING A BOOKMARK

CREATING A BOOKMARK

Having a bookmark in a very long webpage is very useful for the user, instead of scrolling

and trying to find a certain line, bookmarking can help users to save time by having an accessible link provided.

You can give a certain line an "id" tag then to call the id tag simply use the "a href" tag to create a bookmark redirect.

Source Code:
< h1 id: A1 > Sample id < / h1 >
< a href = "#A1">Sample name here< / a >
